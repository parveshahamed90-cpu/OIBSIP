# Unemployment Analysis with Python

## Oasis Infobyte Data Science Internship (OIBSIP)

### Project Overview

This project analyzes unemployment trends in India using Python. The objective is to understand how unemployment rates vary across regions and how the COVID-19 pandemic influenced employment patterns.

### Dataset

The analysis uses unemployment datasets containing:

* Region
* Date
* Estimated Unemployment Rate (%)
* Estimated Employed Population
* Labour Participation Rate (%)
* Area (Rural/Urban)

### Tools and Technologies

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

### Analysis Performed

1. Data Cleaning and Preprocessing
2. Handling Missing Values
3. Regional Unemployment Analysis
4. Employment Trend Visualization
5. COVID-19 Impact Analysis
6. Labour Participation Rate Analysis
7. Comparison of Rural and Urban Employment

### Key Findings

* Significant changes in unemployment rates were observed during the COVID-19 period.
* Some regions experienced higher unemployment than others.
* Labour participation and employment levels showed noticeable fluctuations during lockdown periods.
