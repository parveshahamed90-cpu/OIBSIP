# Email Spam Detection with Machine Learning

## Oasis Infobyte Data Science Internship (OIBSIP)

### Objective

The objective of this project is to build a machine learning model capable of classifying messages as Spam or Ham (Not Spam).

### Dataset

The dataset contains thousands of labeled messages categorized as spam or ham.

### Technologies Used

* Python
* Pandas
* NumPy
* Scikit-Learn
* Matplotlib
* Seaborn

### Machine Learning Algorithm

Multinomial Naive Bayes

### Project Workflow

1. Data Loading
2. Data Cleaning
3. Text Vectorization using TF-IDF
4. Train-Test Split
5. Model Training
6. Prediction
7. Performance Evaluation

### Results

The model successfully classified spam and non-spam messages with high accuracy.
